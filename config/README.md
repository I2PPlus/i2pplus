# Static Analysis Configuration (`config/`)

SpotBugs (formerly FindBugs) configuration and report stylesheets.

## Files

- `spotbugs-exclude.xml` - Exclusion rules for third-party libraries and known false positives
- `spotbugs.xsl` - XSLT stylesheet for formatting SpotBugs HTML reports
- `findbugs.xsl` - Legacy FindBugs XSLT stylesheet (FindBugs was renamed to SpotBugs)
