# Portable I2P+ Distribution

Default configuration files for portable I2P+ distribution.

## Included Configs

- `configs/router.config` - Router settings
- `configs/clients.config` - Client applications
- `configs/i2ptunnel.config` - I2PTunnel settings
- `configs/logger.config` - Logging configuration
- `configs/systray.config` - System tray settings
- `configs/wrapper.config` - Wrapper (procd) settings
- `configs/webapps.config` - Web application settings
- `configs/addressbook/config.txt` - Addressbook configuration
