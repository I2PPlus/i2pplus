• Başlamaq
    Siz yenidən I2P+ başladığınızda, tərəflər hissəsində olan "Aktiv Qonşular" sayısı növbəti bir neçə dəqiqə ərzində artmalı və "Xidmət Tünelləri" bölməsində "Birgə Məlumatlar" adlı "Yerli Müstəqil" də göstərilməlidir, həmçinin I2P-in tənzimlənmə şəklinə görə digər klient və serverlər də görünməlidir. Bu "Yerli Məqsədlər", I2P şəbəkəsinə müxtəlif portlar vasitəsilə xidmətlərə birbaşa əlaqələr təşkil edir və torrent, e-poçt, proxy və digər xidmətlərin işləməsinə imkan yaradır. Şəbəkə Bazası yerli bilindi klientləri göstərir. Həmçinin aktiv tünelləri və onların vəziyyətini izləyə, qonşu əlaqələri və qonşu performans profillərini yoxlaya bilərsiniz.
  • Şəbəkə İnteqrasiyası
    İlk dəfə I2P+ başladığınızda bir neçə dəqiqə ərzində şəbəkəyə inteqrasiya (daxil etmə) oluna bilər və optimal inteqrasiya üçün əlavə qonşular tapa bilər, buna görə lütfən sabirli olun. Yerli Müstəqil adlananın yanında yaşıl ulduz işarəsi göstərildikdən sonra, I2P vasitəsilə etdiyiniz bir çox işlər ola bilər. Aşağıda onlardan bəziləri ilə tanış olmağa çalışacağıq.
  • Şəbəkə Vəziyyəti
    I2P+ şəbəkə əlaqəsinizi yoxladıqdan sonra, tərəflərdəki "Şəbəkə Vəziyyəti" göstəricisi "Tamam" olmalıdır. Əgər "Firewall" göstərirsə, I2P+ işləyəcək lakin performans optimal olmayacaq və şəbəkəyə özünüzün payınızi etmək çətinləşəcəkdir. Optimal iş üçün, xahiş edirik xarici I2P+ portnuz firewall və İSP tərəfindən təqdim edilən ruterdən açın. Status mesajları barədə əlavə məlumat İstifadəçi İnterfeysi kömək səhifəsində mövcuddur.
  • Katib Sadələşdirməsi
    I2P+ .i2p ünvanlarını avtomatik həll etmək üçün, əsasən məlumatlı xidmətlərin DNS tərcüməsini təmin edən xidmətlərə abunə olmalıdır. Əgər I2P-i yenidən quraşdırırsınızsa, artıq kifayət qədər katib abunələri tənzimlənmişdir; quraşdırma dan sonra katibiniz özü aktiv xidmətlərin ünvanları ilə doldurulacaq. Əgər I2P-dən yeniləşdirirsinizsə, əlavə abunələr əlavə etmək üçün katib bölməsində olan Sual-Cavab baxın.
  • Brauzeri Konfiraqsiyası
    Şəbəkədə saytları araşdırmaq üçün, brauzerinizi 4444 portu ilə konfiqurasiyaladılan standart HTTP proksi xidmətindən istifadə etməliyik. Bu, həm də kömür şəbəkə saytlarının anonim araşdırılmasını təmin edən "xariciproksi" xidmətini təqdim edir. Konfiraqsiya səhifəsinə daxil olan Firefox brauzerini tanzimləmə üçün kiçik göstəriş var. Yaxud aşağıda çoxbrauzer tanzimləmə göstərişinə keçid verilir. Katib abunələrini və brauzerinzin konfiraqsiyasını tanzimlədikdən sonra notbob.i2p saytı şəbəkədə mövcud məlumatları araşdırmaq üçün yaxşı bir seçimdir.

I2P xidmətləri

  • Görünməz İnternet və Açıq Web Araşdırma
    I2P-də I2P-də yerləşdirilmiş saytları ("iip saytları") və ya standart şəbəkəni anonim şəkildə I2P-in daxilindəki proksi vasitəsilə araşdıra bilərsiniz. Brauzierinizi 127.0.0.1:4444 portunda HTTP proksini istifadə etmək üçün konfiraseq edin, sonra gizli xidmət saytlarına və ya adi http:// formatli saytlara daxil olun. Standart internetə xariciproksinin girişini söndurmək və ya dəyişmək üçün, "Xariciproksilər" bölməsindən "false.i2p" və ya "purokishi.i2p" üçün proxy qoşulmasını silin və ya dəyişin.
  • Anonim Elektron Poçt
    Postmanın I2P əsaslı poçt sistemi I2P-in daxilindəki poçt kliyentindən (Susimail) və ya standart poçt kliyentindən istifadə edilə bilər. Poçt darvazəsi normal internet ünvanlarından poçt almağa imkan verir; @mail.i2p ünvanınız həm də @i2pmail.org vasitəsilə əlaqələndirilə bilər. Hesab açmaq üçün hq.postman.i2p saytına keçin.
  • Anonim Fayl Ötürüşü
    I2PSnark I2P-ə inteqrasiya olunub və anonim, şifrəli torrent ötürüşünü təmin edir. Həmçinin, BiglyBT (Azureus/Vuze-nin davamı) I2P üzərindən torrentləri dəstəkləyən bir torrent kliyenti, və XD go dilində yazılmış yeganə I2P torrent kliyenti mövcuddur. Əlavə xidmətlərə cake.i2p və fs.i2p kimi brauzer əsaslı fail yerləşdirmə xidmətləri daxildir.
  • Anonim Çat
    IRC kliyentinizi (HexChat kimi) başladın və sunucuya 127.0.0.1:6668 portunda qoşuldunuz. IRC kliyentinizə proksi tənzimləməyisinizə ehtiyac yoxdur; I2P sizə lokal IRC tuneli (#saltr və #i2p-dev kimi kanallara qoşula biləcəyiniz) təqdim edir. Həmçinin Github-də I2P əsaslı ani mesajlaşma proqramı mövcuddur.
  • Anonim Şifrəli Web Yerləşdirməsi
    I2P öz saytınızı I2P şəbəkəsində yerləşdirməyə imkan verən web serveri ilə gəlir: http://127.0.0.1:7658/ ünvanında dinləyən Jetty nümunəsi. I2P Web serveri tunelini başladıqdan sonra saytiniz digərləri üçün görünəcək ((amma tapılmayacaq))dir. .i2p domaini qeydiyyati ilə bağlı kömək kömək bölməsində göstərilir.
  • I2P üçün Pluginlər
    Bir neçə plugin üçün doğrudan yükləmə keçidləri əldə etmək üçün zzz-ın plugin səhifəsinə girin. Əgər inkişafçısınızsa, öz pluginlərinizi yazmaq üçün dilə asılı olmayan bir çatı təqdim edilir.

İnkişafçılar və Təşəkkülçülər üçün Məlumat

  • Xətaları Bəyan Et
    Xəta barədə məlumat vermək üçün, github.com/I2PPlus/i2pplus/issues-da bilet açın. İnkişafçı münasibətləri ilə bağlı müzakirələr üçün, zzz-ın inkişafçı forumlarına və ya I2P şəbəkəsinin IRC kanalına (#i2p-dev) daxil olun.
  • I2P üzərində İnkişaf
    Əgər inkişafçısınızsa, öz I2P proqramlarınızı yaza və ya tətbiqinizin təhlükəsizliyini I2P şəbəkəsindən istifadə etməklə artıra bilərsiniz. Həm də UDP və TCP protokolları təmin edilir və canlı strem də mümkündür. Həmçinin Java API-sinə tam çıxışımız var və əsas I2P kitabxanaları Apache Maven depozitariyasında yerləşdirilir. İnkişafçılar GitHub-da (I2P) və ya Gitlab-da (I2P+) mənbələri gözə bilər və Travis vasitəsilə (I2P) və ya Gitlab vasitəsilə (I2P+) davam edən inteqrasiya prosesini izləyə bilər. Mənbədən tərxiniJava SDK və Apache Ant-a ehtiyac var.
  • Qoşulun!
    I2P məhz toplumu üzvlərinin ödəməsiz könüllü iştirakı ilə inkişaf etdirilir və saxlanılır. İştirak vasitələri barədə əlavə məlumat rasmi saytın könüllülər səhifəsində verilir.
  • I2P+
    I2P+ haqqında əlavə məlumat üçün skank.i2p və ya i2pplus.github.io-ya baxın, yaxud I2P IRC şəbəkəsində #saltR kanalına qoşulun. I2P+ üçün yüngülləşdirmələr xoş gəlinir, lütfən Gitlab vasitəsilə müraciət ərizələri göndərin.
