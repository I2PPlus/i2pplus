• Έναρξη λειτουργίας
    Αν μόλις ξεκινήσατε το I2P+, ο αριθμός των Ενεργών Ομοτίμων που αναγράφεται κάτω από την ενότητα Ομότιμοι στη γραμμή πλοήγησης θα αρχίσει να αυξάνεται τα επόμενα λεπτά, επίσης θα δείτε έναν Τοπικό Προορισμό με όνομα Κοινόχρηστοι Πελάτες καταχωρημένο κάτω από την ενότητα Σήραγγες Υπηρεσιών, και ενδεχωμένως κι άλλους πελάτες και διακομιστές ανάλογα με τον τρόπο που έχει ρυθμιστεί το I2P. Οι Τοπικοί Προορισμοί παρέχουν συνδέσεις σε υπηρεσίες διαφόρων θυρών στο δίκτυο I2P, ενεργοποιώντας το BitTorrent, την ηλεκτρονική αλληλογραφία, τον ιστοπρόξυ και άλλες υπηρεσίες να λειτουργούν. Η Βάση Δεδομένων Δικτύου σας υποδεικνύει όλους τους γνωστούς τοπικά ομότιμους στο δίκτυο. Μπορείτε επίσης να παρακολουθείτε υπάρχουσες συνδέσεις ομοτίμων, να εξετάζετε προφίλ απόδοσης ομοτίμων και να παρατηρείτε ενεργές σήραγγες και την κατάστασή τους.
  • Ένταξη στο Δίκτυο
    Την πρώτη φορά που θα ξεκινήσετε το I2P+ ίσως χρειαστεί λίγα λεπτά για να εκκινήσει η διαδικασία bootstrap (ενσωμάτωσης) σας στο δίκτυο και να εντοπιστούν επιπρόσθετοι ομότιμοι για τη βελτιστοποίηση της ένταξής σας, οπότε παρακαλώ θέστε λίγη υπομονή. Όταν δείτε πράσινα αστέρια δίπλα από τους Τοπικούς Προορισμούς σας κάτω από τις Σήραγγες Υπηρεσιών στη γραμμή πλοήγησης, υπάρχουν πολλές δυνατότητες που μπορείτε να κάνετε με το I2P και στη συνέχεια σας συστήνουμε μερικές απ' αυτές.
  • Κατάσταση Δικτύου
    Μόλις ο I2P+ έχει χρόνο να δοκιμάσει τη συνδεσιμότητα δικτύου σας, ο δείκτης κατάστασης δικτύου στη γραμμή πλευράς θα πρέπει να εμφανίζει "OK". Αν η κατάσταση δείχνει "Περιφραγμένο", ο I2P+ θα λειτουργεί ακόμα, αν και η απόδοση του δικτύου δεν θα είναι βέλτιστη και η ικανότητά σας να συνεισφέρετε ζώνη εύρους στο δίκτυο θα εμποδίζεται. Για βέλτιστη απόδοση, σας παρακαλούμε να εξετάσετε το άνοιγμα της εξωτερικής πύλης I2P+ στην περιφραγμένη περιοχή σας και στο δρομολογητή ISP που παρέχεται. Περισσότερες πληροφορίες σχετικά με τα μηνύματα κατάστασης είναι διαθέσιμες στη σελίδα βοήθειας διεπαφής χρήστη.
  • Συνδρομές Βιβλίου Διευθύνσεων
    Γι να μπορεί ο I2P+ να μπορεί να επιλύει αυτόματα τα ονόματες τομέα .i2p, εξαρτάται κυρίως από συνδρομές σε υπηρεσίες που παρέχουν μετάφραση DNS για γνωστούς οικοδεσπότες. Αν εγκαταστήσατε τον I2P+ αντί να αναβαθμίσετε από τον I2P, θα διαθέτετε ήδη επαρκείς συνδρομές βιβλίου διευθύνσεων εξασφαλισμένες μετά την εγκατάσταση, το βιβλίο διευθύνσεων σας θα πληρούται με ονόματα οικοδεσπότες για ενεργές υπηρεσίες στο δίκτυο χωρίς παρέμβαση. Αν αναβαθμίσατε από τον I2P, ελέγξτε το τμήμα συνδρομών βιβλίου διευθύνσεων στην Ενότητα Συχνών Ερωτήσεων για βοήθεια στην προσθήκη επιπλέον συνδρομών.
  • Διαμόρφωση Περιηγητή
    Για να περιηγηθείτε σε ιστοσελίδες στο δίκτυο, θα χρειαστεί να διαμορφώσετε τον περιηγητή σας για να χρησιμοποιεί τον προεπιλεγμένο διαμεσολαβητή HTTP που έχει διαμορφωθεί στην πύλη 4444, ο οποίος παρέχει επίσης έναν outproxy που επιτρέπει ανώνυμη περιήγηση σε διαδικτυακούς τόπους. Ένας μικρός οδηγός διαμόρφωσης του Firefox παρέχεται στη σελίδα Ρυθμίσεων στην ενότητα βοήθειας, ή ανατρέξτε στο τμήμα κάτωθι για σύνδεσμο στον οδηγό διαμόρφωσης πολλαπλών περιηγητών. Μόλις διαμορφώσετε τον περιηγητή και τις συνδρομές βιβλίου διευθύνσεων σας, το notbob.i2p είναι ένας εξαιρετικός τόπος για την ανακάλυψη περιεχομένου στο δίκτυο.

Υπηρεσίες στο I2P

  • Αόρατο Διαδίκτυο & Δημόσια Περιήγηση στον Ιστό
    Στο I2P μπορείτε να προσπελάσετε ιστοσελίδες φιλοξενούμενες στο I2P ("eepsites") ή να περιηγηθείτε ανώνυμα στον κανονικό ιστό μέσω του ενσωματωμένου διαμεσολαβητή ιστού του I2P (outproxy). Διαμορφώστε τον περιηγητή σας για να χρησιμοποιεί τον HTTP διαμεσολαβητή στη διεύθυνση 127.0.0.1 λιμάνι 4444, στη συνέχεια περιηγηθείτε σε μια κρυφή ιστοσελίδα υπηρεσίας ή σε μια κανονική, μη κρυπτογραφημένη διεύθυνση http://.
  • Ανώνυμο Ηλεκτρονικό Ταχυδρομείο
    Το σύστημα ηλεκτρονικού ταχυδρομείου του Postman το οποίο βασίζεται στο I2P μπορεί να προσπελαστεί είτε μέσω του ενσωματωμένου πελάτη ηλεκτρονικού ταχυδρομείου του I2P (Susimail) είτε μέσω οποιουδήποτε πρότυπου πελάτη ηλεκτρονικού ταχυδρομείου. Μια πύλη ηλεκτρονικού ταχυδρομείου επιτρέπει τη λήψη ηλεκτρονικού ταχυδρομείου από κανονικές διευθύνσες διαδικτύου· η διεύθυνσή σας με τη μορφή @mail.i2p είναι επίσης επαφή μέσω @i2pmail.org. Για έναν λογαριασμό επισκεφθείτε τη διεύθυνση hq.postman.i2p.
  • Ανώνυμη ΜετΑρχείων
    Το I2PSnark είναι ενσωματωμένο στο I2P, παρέχοντας ανώνυμες, κρυπτογραφημένες μεταφορές BitTorrent. Εναλλακτικά υπάρχει το BiglyBT (ο διάδοχος του Azureus/Vuze), ένας πελάτης BitTorrent ο οποίος υποστηρίζει μεταφορές torrent μέσω I2P, και το XD, ένας πελάτης BitTorrent γραμμένος σε go και ειδικά για I2P. Επιπλέον υπηρεσίες περιλαμβάνουν φιλοξενία αρχείων βασισμένη στον περιηγητή όπως τα cake.i2p και fs.i2p.
  • Ανώνυμη Συνομιλία
    Εκκινήστε τον πελάτη IRC σας (π.χ. HexChat) και συνδεθείτε στον διακομιστή στη διεύθυνση 127.0.0.1 λιμάνι 6668. Δεν χρειάζεται να ρυθμίσετε έναν διαμεσολαβητή στον πελάτη IRC σας· το I2P σας παρέχει ένα τοπικό ισχύον IRC (ρυθμισμένο στον πελάτη IRC σας ως διακομιστής IRC ή απομακρυσμένος οικοδεσπότης). Μερικά από τα διαθέσιμα κανάλια που θα ήθελες να ενωθείς είναι τα #saltr και #i2p-dev. Υπάρχει επίσης μια διαπλατφορμική εφαρμογή ανταλλαγής ασαφών μηνυμάτων μέσω I2P, διαθέσιμη στο Github.
  • Ανώνυμη Κρυπτογραφημένη Φιλοξενία Ιστού
    Το I2P περιλαμβάνει έναν διακομιστή ιστού για να φιλοξενήσετε την ιστοσελίδα σας στο δίκτυο I2P: μια Jetty υλοποίηση που ακούει στη διεύθυνση http://127.0.0.1:7658/. Μετά την εκκίνηση της προ-ρυθμισμένης Σήραγγας I2P Webserver, η ιστοσελίδα σας θα είναι ορατή (αλλά όχι ανιχνεύσιμη) από άλλους. Οδηγίες για τη ρύθμιση της ιστοσελίδας σας και καταγραφή ενός τομέα .i2p παρέχονται στην ενότητα βοήθειας.
  • Πρόσθετα για το I2P
    Επισκεφθείτε τη σελίδα πρόσθετων του zzz για άμεσους συνδέσμους λήψης αρκετών προσθέτων. Αν είστε προγραμματιστής, παρέχεται ένα πλαίσιο ανεξάρτητο γλώσσας για την ανάπτυξη των δικών σας προσθέτων.

Πληροφορίες για Προγραμματιστές & Συνεισφέροντες

  • Αναφορά Σφαλμάτων
    Για να αναφέρετε ένα σφάλμα, παρακαλώ υποβάλετε ένα εισιτήριο στο github.com/I2PPlus/i2pplus/issues. Για συζητήσεις ανάπτυξης, επισκεφθείτε τα φόρουμ αναπτυκτών του zzz ή το κανάλι ανάπτυξης (#i2p-dev) στο IRC δίκτυο του I2P.
  • Ανάπτυξη στο I2P
    Αν είστε προγραμματιστής, μπορεί να σκεφτείτε να δημιουργήσετε δικές σας εφαρμογές I2P ή να ενισχύσετε την ασφάλεια της εφαρμογής σας εκμεταλλευόμενοι το δίκτυο I2P. Τόσο το πρωτόκολλο UDP όσο και το πρωτόκολλο TCP παρέχονται, και η πραγματικού χρόνου μετάδοση είναι επίσης δυνατή. Παρέχουμε πλήρη πρόσβαση στην Java API, και οι βασικές βιβλιοθήκες I2P φιλοξενούνται στο αποθετήριο Apache Maven. Οι προγραμματιστές μπορούν να ρίξουν μια ματιά στον πηγαίο κώδικα στο Github (I2P) ή Gitlab (I2P+), και να παρακολουθούν την τρέχουσα κατάσταση κτίσης μέσω της πλατφόρμας συνεχούς ολοκλήρωσης Travis (I2P) ή μέσω Gitlab (I2P+). Απαιτείται ένα Java SDK και το Apache Ant για την ολοκλήρωση από πηγαίο κώδικα.
  • Συμμετάσχετε!
    Το I2P αναπτύσσεται και συντηρείται κυρίως μέσω εθελοντικής συμμετοχής μελών της κοινότητας, χωρίς χρηματοδότηση. Περισσότερες πληροφορίες για το πώς μπορείτε να συμμετάσχετε διατίθενται στη σελίδα εθελοντών της επίσημης ιστοσελίδας.
  • I2P+
    Για περισσότερες πληροφορίες σχετικά με το I2P+, παρακαλώ επισκεφθείτε τις διευθύνσεις skank.i2p ή i2pplus.github.io, ή ενταχθείτε στο κανάλι #saltR στο IRC δίκτυο του I2P. Τα αιτήματα βελτίωσης του I2P+ είναι ευπρόσδεκτα· παρακαλώ υποβάλετε αιτήματα σύνδεσης μέσω Gitlab.
