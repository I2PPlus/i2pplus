The I2P+ Windows Service is now installed and should be running automatically. To manage the service startup properties, run services.msc from the start menu or a command line, and then adjust the I2P+ service accordingly, or configure from within the Router Console.

Additionally, shortcuts are provided to stop and start the service, though in normal operation you should use the console to stop the service as this will allow the router to honor its networking obligations before shutting down or restarting.

In the unlikely event the Windows service fails to install properly, you can run I2P+ as a standard application via the i2p.exe file in your application folder, usually found at: C:\Program Files\i2p\
