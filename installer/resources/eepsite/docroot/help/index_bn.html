I2P বেনামী ওয়েব সার্ভার

I2P নেটওয়ার্কে ওয়েবসাইট হোস্টিং সম্পর্কে ভূমিকা এবং আরও তথ্যের জন্য, অনুগ্রহ করে আপনার কনসোলে সাহায্য বিভাগটি দেখুন:

  • নতুন ব্যবহারকারীর নির্দেশিকা
  • I2P-এ ওয়েব হোস্টিং
I2P ওয়েব সার্ভার রুট ডিরেক্টরির (docroot) অবস্থান
%APPDATA%\I2P\eepsite\docroot\
/Users/(ব্যবহারকারীর নাম)/Library/Application Support/i2p
স্ট্যান্ডার্ড Java ইনস্টলেশন ➜ ~/.i2p/eepsite/docroot/
প্যাকেজ/রিপোজিটরি ইনস্টলেশন ➜ /var/lib/i2p/i2p-config/eepsite/docroot/

আপনার নতুন ওয়েবসাইট বিশ্বের কাছে ঘোষণা করার আগে, আপনার কিছু কন্টেন্ট যোগ করা উচিত। সার্ভারের রুট ডিরেক্টরিতে গিয়ে index.html রিডাইরেক্ট পৃষ্ঠাটি আপনার নিজস্ব কন্টেন্ট দিয়ে প্রতিস্থাপন করুন। যদি একটি মৌলিক সাইটের জন্য টেমপ্লেটের প্রয়োজন হয়, তাহলে নির্দ্বিধায় এই পৃষ্ঠাটি অভিযোজিত করুন। কন্টেন্ট সম্পাদনা করার পর যদি আপনি এই পৃষ্ঠায় ফিরে আসেন, তাহলে আপনার ব্রাউজারের ক্যাশে সাফ করার চেষ্টা করুন।
