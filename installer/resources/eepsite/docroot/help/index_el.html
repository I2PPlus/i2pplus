Σερβερ Ανώνυμης Ιντερνέτ Διακομιστής I2P

Για μια εισαγωγή και περισσότερες πληροφορίες σχετικά με τη διαχείριση ιστοσελίδων στο δίκτυο I2P, αναφερθείτε στην ενότητα βοήθειας του συστήματός σας:

  • Οδηγός Νέου Χρήστη
  • Web Hosting στο I2P
Τοποθεσία της βάσης δεδομένων του διακομιστή I2P (docroot)
%APPDATA%\I2P\eepsite\docroot\
/Users/(όνομα χρήστη)/Library/Application Support/i2p
Κανονική εγκατάσταση Java ➜ ~/.i2p/eepsite/docroot/
Εγκατάσταση από πακέτο/αποθεματικό ➜ /var/lib/i2p/i2p-config/eepsite/docroot/

Πριν ανακοινώσετε τον νέο σας ιστότοπο στον κόσμο, πρέπει να προσθέσετε περιεχόμενο. Πηγαίνετε στη βάση δεδομένων του διακομιστή που αναφέρεται παραπάνω και αντικαταστήστε τη σελίδα ανακατεύθυνσης index.html με το δικό σας περιεχόμενο. Εάν χρειάζεστε ένα πρότυπο για ένα βασικό ιστότοπο, αισθάνεστε ελεύθεροι να χρησιμοποιήσετε αυτήν τη σελίδα. Εάν επιστρέψετε σε αυτήν τη σελίδα μετά την επεξεργασία του περιεχομένου, δοκιμάστε να καθαρίσετε τον αποθηκευτικό χώρο του προγράμματος περιήγησης σας.
