I2P Anonymní Webový Server

Pro úvod a další informace o hostování webových stránek v síti I2P se prosím odkazujte na sekci nápovědy ve vašem konzoli:

  • Průvodce pro nové uživatele
  • Webové hostování na I2P
Umístění kořenové složky webového serveru I2P (docroot)
%APPDATA%\I2P\eepsite\docroot\
/Users/(uživatelské jméno)/Library/Application Support/i2p
Standartní instalace Java ➜ ~/.i2p/eepsite/docroot/
Instalace z balíčku/repozitáře ➜ /var/lib/i2p/i2p-config/eepsite/docroot/

Než oznámíte světu váš nový web, měli byste přidat nějaký obsah. Pojděte do uvedené kořenové složky serveru a nahraďte stránku přesměrování index.html svým vlastním obsahem. Pokud potřebujete šablonu pro základní web, neváhejte upravit tuto stránku. Pokud se vrátíte na tuto stránku po úpravách obsahu, zkuste vyčistit cache vašeho prohlížeče.
