I2P 익명 웹 서버

I2P 네트워크에서 웹사이트 호스팅에 대한 소개 및 자세한 정보는 콘솔의 도움 섹션을 참조하십시오:

  • 새 사용자 가이드
  • I2P에서 웹 호스팅
I2P 웹 서버 루트 디렉토리(docroot) 위치
%APPDATA%\I2P\eepsite\docroot\
/Users/(사용자명)/Library/Application Support/i2p
표준 Java 설치 ➜ ~/.i2p/eepsite/docroot/
패키지/저장소 설치 ➜ /var/lib/i2p/i2p-config/eepsite/docroot/

새 웹사이트를 세상에 알기 전에 먼저 콘텐츠를 추가해야 합니다. 위에 나열된 서버의 루트 디렉토리로 이동하여 index.html 리디렉션 페이지를 자신의 콘텐츠로 바꿉니다. 기본 웹사이트의 템플릿이 필요한 경우 이 페이지를 자유롭게 참고하세요. 콘텐츠를 편집한 후 이 페이지로 돌아와서 브라우저의 웹 캐시를 지우는 것을 고려해 보세요.
