DNS message parsing and serialization.

This package contains the core classes for representing and manipulating DNS protocol messages. It provides both low-level parsing of raw DNS packet data and high-level abstractions for working with DNS messages in a structured way.

Key components:

  • DNS message structure (header, questions, answers, authority, additional)
  • Message parsing from wire format
  • Message serialization to wire format
  • EDNS(0) extension support
  • DNS over TCP/UDP transport handling

The classes in this package form the foundation for all DNS operations in MiniDNS, providing the ability to construct, parse, and validate DNS protocol messages according to RFC 1035 and related specifications.
