DNS record types and constants for the MiniDNS library.

This package contains enumerations and constant definitions for various DNS record types, classes, and opcodes used throughout the MiniDNS implementation. These constants provide type-safe references to standard DNS protocol values.

Key components include:

  • Record type constants (A, AAAA, MX, TXT, etc.)
  • DNS class constants (IN, CH, etc.)
  • Response code definitions
  • Protocol opcode constants
