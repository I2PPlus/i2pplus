DNS resource record types and implementations.

This package contains classes representing various DNS resource record types, providing structured access to record data and functionality for parsing and serialization of different record formats.

Supported record types include:

  • Address Records: A (IPv4), AAAA (IPv6)
  • Name Server Records: NS, SOA
  • Mail Records: MX, SPF
  • Text Records: TXT, DKIM
  • Security Records: DNSKEY, DS, RRSIG
  • Infrastructure Records: CNAME, DNAME
  • Service Records: SRV
  • Experimental Records: NSEC, NSEC3

Each record type provides type-safe access to its specific data fields and handles the appropriate parsing and serialization according to the relevant RFC specifications.
