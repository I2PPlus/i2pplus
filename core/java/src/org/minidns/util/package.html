Utility classes and helper functions for MiniDNS.

This package contains various utility classes and helper methods that support the core DNS functionality in MiniDNS. These utilities provide common operations used throughout the library, including data conversion, validation, and convenience methods.

Key utilities include:

  • Hexadecimal and binary data conversion
  • Byte array manipulation helpers
  • Input/output stream utilities
  • String formatting and parsing helpers
  • Collections utilities for DNS data structures
  • Exception handling and error reporting
