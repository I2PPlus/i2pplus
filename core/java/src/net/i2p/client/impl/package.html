Internal implementation of the I2P client SDK, providing the client side of the I2CP protocol for applications communicating through I2P.

This package contains:

  • I2PSession implementation and management
  • I2CP protocol message handling
  • Client connection lifecycle management
  • Session configuration and state handling
  • Message sending and receiving logic

These classes implement the client side of I2CP and are not to be used externally. Subject to change and not part of the public API. Moved from net.i2p.client in 0.9.21 to separate implementation from interface.
