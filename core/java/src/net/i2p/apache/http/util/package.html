HTTP utility classes from Apache HttpCore 4.4.1, providing HostnameVerifier and related HTTP utilities.

This package provides:

  • HostnameVerifier for SSL certificate validation
  • HTTP utility classes and helpers
  • Connection management utilities

Extracted from Apache HttpCore 4.4.1 with minimal modifications to remove additional Apache dependencies while maintaining core functionality. Provides essential HTTP utilities for I2P's networking layer.

Apache 2.0 license. Moved to net.i2p.apache.* packages in release 1.8.0 (API 0.9.54).
