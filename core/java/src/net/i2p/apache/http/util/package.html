HTTP utility classes from Apache HttpCore 4.4.1, providing argument validation and related HTTP utilities.

This package provides HTTP utility components:

  • Args: Argument validation and assertion utilities for HTTP operations
  • HTTP helpers: Core utility classes for HTTP protocol handling

Extracted from Apache HttpCore 4.4.1 with minimal modifications to remove additional Apache dependencies while maintaining core functionality. Provides essential HTTP utilities for I2P's networking layer and HTTP client operations.
