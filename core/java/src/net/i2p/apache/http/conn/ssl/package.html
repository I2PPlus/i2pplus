SSL/TLS socket factory from Apache HttpClient 4.4.1, providing I2PSSSLSocketFactory for secure HTTP connections.

This package provides:

  • I2PSSSLSocketFactory for SSL connections
  • Custom SSL context management
  • Certificate validation helpers

Extracted from Apache HttpClient 4.4.1 with minimal modifications to remove additional Apache dependencies. I2PSSSLSocketFactory provides I2P-specific SSL socket factory for HTTP client operations, integrating with I2P's certificate validation and trust management.
