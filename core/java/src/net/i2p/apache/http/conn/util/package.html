HTTP connection utilities from Apache HttpClient 4.4.1, providing connection management and verification classes.

This package provides:

  • Connection pooling utilities
  • SSL/TLS connection helpers
  • Hostname verification for secure connections

Extracted from Apache HttpClient 4.4.1 with minimal modifications to remove additional Apache dependencies while maintaining core connection functionality. Used by I2P for HTTP client operations requiring secure connections.

Apache 2.0 license. Moved to net.i2p.apache.* packages in release 1.8.0 (API 0.9.54).
