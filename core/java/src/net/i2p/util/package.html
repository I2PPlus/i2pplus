Core utility classes and helper functions used throughout the I2P router and applications.

This package provides essential functionality including:

  • String manipulation and formatting utilities
  • Collection helpers and data structures
  • File I/O and stream utilities
  • Mathematical and conversion functions
  • Time and date handling
  • Network address utilities
  • Concurrency and synchronization helpers

These utilities are designed to be lightweight, efficient, and widely reusable across the I2P codebase. Many of these classes are static utility classes that provide commonly needed functionality without requiring instantiation.
