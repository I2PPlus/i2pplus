Java Security Provider implementation for I2P cryptographic algorithms.

This package provides a {@link java.security.Provider} implementation that registers I2P's cryptographic algorithms with the Java Cryptography Architecture (JCA). The provider enables seamless integration of I2P algorithms with standard Java security APIs.

Supported Algorithms:

  • EdDSA: Edwards-curve Digital Signature Algorithm (Ed25519)
  • Future Support: Additional I2P algorithms may be added in subsequent versions

Usage:

This provider is primarily needed when using runtime registration with: 

    Security.addProvider(new I2PProvider());

Once registered, algorithms can be accessed through standard JCA APIs such as:

  • {@link java.security.Signature} for digital signatures
  • {@link java.security.KeyPairGenerator} for key generation
  • {@link java.security.KeyFactory} for key conversion

When to Use:

  • Applications requiring JCA compatibility with I2P algorithms
  • Integration with existing Java security frameworks
  • Standardized algorithm access patterns

Since 0.9.15. Other I2P algorithms may be added later.
