Parameter specifications and algorithm configuration for X25519 operations.

This package provides {@link java.security.spec.AlgorithmParameterSpec} implementations that define the parameters and configuration for X25519 Diffie-Hellman key exchange. These specifications enable proper initialization and configuration of X25519 operations through the Java Cryptography Architecture (JCA).

Key Components:

  • X25519Spec: Parameter specification for X25519 key operations
  • Configuration: Standardized parameter definitions for algorithm initialization
  • JCA Integration: Compatibility with Java security provider framework

These specifications are used when:

  • Initializing X25519 key generators
  • Configuring key agreement operations
  • Providing algorithm parameters to JCA instances

Since 0.9.38.
