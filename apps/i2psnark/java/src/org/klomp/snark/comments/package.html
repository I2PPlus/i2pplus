Data structures to support ut_comment, since 0.9.31.

This package provides implementation for the uTorrent comment extension (ut_comment), which allows torrent creators to include comments and metadata in torrent files. These comments can contain information about the torrent content, creation details, or other relevant metadata.

Key features include:

  • Parsing and storage of torrent comments
  • Support for UTF-8 encoded comments
  • Integration with I2PSnark's torrent management system
  • Display of comments in the web interface
  • Comment preservation during torrent operations

The ut_comment extension enhances the torrent sharing experience by allowing content creators to provide additional context and information about their torrents, making it easier for users to understand torrent contents and purposes.
