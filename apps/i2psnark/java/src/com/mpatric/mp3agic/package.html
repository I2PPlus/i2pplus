MP3 file processing and metadata extraction utilities.

This package provides functionality for reading and processing MP3 audio files, including metadata extraction, frame analysis, and audio stream information. It's used by I2PSnark for handling torrent files that contain MP3 content.

Key features:

  • ID3 tag parsing (v1, v2.2, v2.3, v2.4)
  • Audio frame analysis
  • Bitrate and duration calculation
  • Album artwork extraction
  • Metadata normalization
