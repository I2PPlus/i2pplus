# SusiDNS (`susidns/`)

Web UI for the address book and local naming service. Provides DNS management (address book entries, subscriptions, resolution) for the I2P network.

## Packages

- `i2p.susi.dns` - Addressbook, config, naming service, and subscription beans

## Building

```bash
# Gradle
./gradlew :apps:susidns:jar

# Ant (from repo root)
ant buildSusiDNS
```
