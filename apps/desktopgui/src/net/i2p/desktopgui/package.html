Desktop graphical interface for managing I2P router lifecycle from the system tray.

This package provides:

  • System tray icon integration
  • Router start/stop/restart controls
  • Status monitoring and notifications
  • Router console quick access
  • Configuration shortcuts

Desktopgui provides a convenient system tray interface for managing the I2P router without requiring a browser window open. It displays router status, provides quick access to common operations, and shows notifications for important events.
