Router integration and communication for the Desktop GUI.

This package handles the communication between the Desktop GUI application and the I2P router. It provides abstractions for monitoring router status, configuring settings, and receiving notifications from the router.

Key functionality:

  • Router status monitoring
  • Configuration management
  • Event notification handling
  • Service lifecycle management
  • Performance metrics collection

This package enables the Desktop GUI to provide a user-friendly interface for managing and monitoring the I2P router without requiring direct access to router internals.
