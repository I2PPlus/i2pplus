Utility classes used by the JFreeSVG core library. These provide argument validation, shape copying, gradient paint deduplication keys, and fast double-to-string conversion.

Key Components

  • {@link org.jfree.svg.util.Args} - Argument validation (null checks, array length checks)
  • {@link org.jfree.svg.util.GraphicsUtils} - Shape copying and image conversion utilities
  • {@link org.jfree.svg.util.GradientPaintKey} - HashMap key for {@code java.awt.GradientPaint}
  • {@link org.jfree.svg.util.LinearGradientPaintKey} - HashMap key for {@code java.awt.LinearGradientPaint}
  • {@link org.jfree.svg.util.RadialGradientPaintKey} - HashMap key for {@code java.awt.RadialGradientPaint}
  • {@link org.jfree.svg.util.RyuDouble} - Ryu algorithm for fast double-to-string conversion

Related Packages

  • {@link org.jfree.svg} - Core JFreeSVG SVG generation classes
