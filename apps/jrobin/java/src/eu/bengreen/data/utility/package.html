Data utility classes for hashing and time-related algorithms. This package provides utility classes for data processing, including hashing algorithms and time specification parsing used by the JRobin RRD (Round Robin Database) system for time series data storage and analysis.

Key Components

  • {@link eu.bengreen.data.utility.DownSampleImpl} - Downsampling implementation
  • {@link eu.bengreen.data.utility.LargestTriangleThreeBuckets} - Time bucket algorithm
  • {@link eu.bengreen.data.utility.LargestTriangleThreeBucketsTime} - Time-based bucket implementation

Main Features

  • Efficient data downsampling algorithms
  • Time-based data aggregation methods
  • Statistical utility functions
  • Performance-optimized implementations
  • Integration with RRD data structures

Usage Context

These utilities are primarily used by the JRobin RRD system for efficient data storage and retrieval. The downsampling algorithms help reduce data resolution while preserving important trends and patterns in time series data.

Related Packages

  • {@link org.rrd4j} - Main JRobin RRD implementation
  • {@link org.rrd4j.core} - Core RRD functionality
