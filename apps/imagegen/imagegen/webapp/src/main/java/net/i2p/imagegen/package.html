Servlets and services for generating QR codes and identicon images for I2P destinations and identifiers.

This package provides:

  • QR code generation for I2P destinations
  • Identicon generation for visual identifiers
  • HTTP servlets for image serving
  • Image caching and optimization

These services provide visual representations of I2P destinations, making them easier to identify and share. QR codes allow destinations to be scanned and imported easily, while identicons provide unique visual identifiers for recognition.
