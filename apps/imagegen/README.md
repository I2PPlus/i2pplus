# Image Generation (`imagegen/`)

Servlets for generating QR codes and identicon images for I2P destinations, enabling visual identification and easy sharing of destination addresses.

## Packages

- `net.i2p.imagegen` - QR code, identicon, and random art servlets

## Building

```bash
# Gradle
./gradlew :apps:imagegen:jar

# Ant (from repo root)
ant buildImagegen
```
