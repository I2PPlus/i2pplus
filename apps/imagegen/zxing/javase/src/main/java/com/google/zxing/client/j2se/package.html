Java SE client-side implementation for ZXing barcode library. This package provides Java SE (Standard Edition) specific implementations for barcode generation and rendering, allowing desktop applications to create barcode images using standard Java libraries without requiring Android-specific functionality.

Key Components

  • {@link com.google.zxing.client.j2se.MatrixToImageWriter} - Converts barcode matrices to images
  • {@link com.google.zxing.client.j2se.MatrixToImageConfig} - Configuration for image generation

Main Features

  • Image output in various formats (PNG, JPEG, GIF, BMP)
  • Configurable image dimensions and margins
  • Support for different color schemes
  • Anti-aliasing and image quality options
  • Integration with AWT for desktop applications

Usage Example

// Create a QR code image
BitMatrix matrix = writer.encode(content, BarcodeFormat.QR_CODE);
MatrixToImageConfig config = new MatrixToImageConfig();
BufferedImage image = MatrixToImageWriter.toBufferedImage(matrix, config);
ImageIO.write(image, "png", new File("qrcode.png"));
