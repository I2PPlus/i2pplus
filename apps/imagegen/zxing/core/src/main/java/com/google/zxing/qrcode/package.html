ZXing core QR code implementation.

This package contains the core QR code encoding and decoding functionality from the ZXing (Zebra Crossing) library. It provides comprehensive support for generating and reading QR codes in various formats and with different error correction levels.

Key features:

  • QR code encoding and generation
  • QR code decoding and recognition
  • Error correction implementation
  • Multiple format support
  • Performance optimization

This functionality is used by the I2P image generation app for creating QR codes that can contain I2P destinations, keys, or other data for easy sharing and scanning.
