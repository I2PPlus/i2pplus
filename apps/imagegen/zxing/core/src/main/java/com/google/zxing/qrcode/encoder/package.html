QR code encoding implementation.

This package provides encoding functionality for creating QR codes, including data encoding, error correction generation, and matrix construction. It implements algorithms needed to convert arbitrary data into valid QR code patterns.

Features include:

  • Data mode encoding (numeric, alphanumeric, byte, kanji)
  • Error correction code generation
  • Matrix pattern construction
  • Mask pattern selection and application
  • Version and format information encoding

This encoder is used by the I2P image generation app to create QR codes containing I2P destinations, configuration data, or other information that can be easily shared with mobile devices or other QR code readers.
