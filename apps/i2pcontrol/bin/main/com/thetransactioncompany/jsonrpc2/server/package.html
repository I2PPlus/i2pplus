JSON-RPC 2.0 server implementation for I2P Control interface.

This package provides the server-side components for handling JSON-RPC 2.0 requests from I2P Control clients, enabling remote management and monitoring of I2P router functionality through a standardized web service interface.

Key features include:

  • JSON-RPC 2.0 protocol compliance
  • Request routing and dispatching
  • Error handling and response formatting
  • Integration with I2P router services
