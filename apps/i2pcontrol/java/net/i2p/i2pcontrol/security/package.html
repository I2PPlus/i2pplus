Security components for I2P Control authentication and authorization. This package provides the security infrastructure for the I2P Control service, including authentication token management, SSL certificate handling, and access control.

Key Components

  • {@link net.i2p.i2pcontrol.security.SecurityManager} - Manages authentication tokens and security policies
  • {@link net.i2p.i2pcontrol.security.AuthToken} - Represents an authentication token
  • {@link net.i2p.i2pcontrol.security.KeyStoreProvider} - Provides SSL certificate management

Security Features

  • Token-based authentication with configurable expiration
  • SSL/TLS support for secure communication
  • Password hashing using BCrypt
  • Configurable access control policies

Related Packages

  • {@link net.i2p.i2pcontrol} - Main I2P Control functionality
  • {@link net.i2p.i2pcontrol.servlets} - HTTP servlets and handlers
