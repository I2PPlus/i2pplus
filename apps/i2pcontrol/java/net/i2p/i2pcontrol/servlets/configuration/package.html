Configuration management for I2P Control settings and parameters. This package provides functionality for managing I2P Control configuration, including loading, saving, and accessing configuration parameters that control the behavior of the I2P Control service.

Key Components

  • {@link net.i2p.i2pcontrol.servlets.configuration.ConfigurationManager} - Main configuration management class

Configuration Features

  • Configuration file loading and saving
  • Parameter validation and default values
  • Runtime configuration updates
  • Integration with I2P configuration system

Related Packages

  • {@link net.i2p.i2pcontrol.servlets} - Parent servlet package
  • {@link net.i2p.i2pcontrol} - Main I2P Control functionality
