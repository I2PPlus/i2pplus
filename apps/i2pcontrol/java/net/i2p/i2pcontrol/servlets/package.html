HTTP servlets for handling I2P Control JSON-RPC 2.0 requests. This package contains the main servlet and HTTP handlers for processing I2P Control API requests. It provides the web interface layer that translates HTTP requests into JSON-RPC calls and routes them to the appropriate handlers.

Key Components

  • {@link net.i2p.i2pcontrol.servlets.JSONRPC2Servlet} - Main servlet handling JSON-RPC 2.0 requests
  • {@link net.i2p.i2pcontrol.servlets.configuration.ConfigurationManager} - Manages I2P Control configuration

Subpackages

  • {@link net.i2p.i2pcontrol.servlets.configuration} - Configuration management
  • {@link net.i2p.i2pcontrol.servlets.jsonrpc2handlers} - JSON-RPC method handlers

Related Packages

  • {@link net.i2p.i2pcontrol} - Main I2P Control functionality
  • {@link net.i2p.i2pcontrol.security} - Security and authentication
