JSON-RPC 2.0 handlers for I2P Control API methods. This package contains specific handlers for different JSON-RPC 2.0 method calls in the I2P Control API. Each handler is responsible for processing specific API requests and returning appropriate responses.

Key Handlers

  • {@link net.i2p.i2pcontrol.servlets.jsonrpc2handlers.I2PControlHandler} - Base class for I2P Control handlers
  • {@link net.i2p.i2pcontrol.servlets.jsonrpc2handlers.RouterManagerHandler} - Router management operations
  • {@link net.i2p.i2pcontrol.servlets.jsonrpc2handlers.NetworkSettingHandler} - Network configuration
  • {@link net.i2p.i2pcontrol.servlets.jsonrpc2handlers.RouterInfoHandler} - Router information
  • {@link net.i2p.i2pcontrol.servlets.jsonrpc2handlers.GetRateHandler} - Performance statistics
  • {@link net.i2p.i2pcontrol.servlets.jsonrpc2handlers.AdvancedSettingsHandler} - Advanced configuration
  • {@link net.i2p.i2pcontrol.servlets.jsonrpc2handlers.AuthenticateHandler} - Authentication
  • {@link net.i2p.i2pcontrol.servlets.jsonrpc2handlers.EchoHandler} - Testing/echo functionality

Handler Categories

  • Authentication and security
  • Router management and control
  • Network configuration and monitoring
  • Statistics and performance data
  • Advanced settings and debugging
