I2P Control provides a JSON-RPC 2.0 interface for controlling and monitoring the I2P router. This package contains the main controller and core functionality for the I2P Control service, allowing external applications to interact with the I2P router through a secure API.

Key Components

  • {@link net.i2p.i2pcontrol.I2PControlController} - Main controller for the I2P Control service
  • {@link net.i2p.i2pcontrol.SocketController} - Handles socket connections
  • {@link net.i2p.i2pcontrol.I2PControlVersion} - Version information

Related Packages

  • {@link net.i2p.i2pcontrol.security} - Security and authentication components
  • {@link net.i2p.i2pcontrol.servlets} - HTTP servlets and JSON-RPC handlers
