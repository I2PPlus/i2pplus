The default naming service implementation for the I2P router, using the BlockfileNamingService based on the Metanotion BlockFile Database.

This package provides:

  • Host name to I2P destination resolution
  • Local hostname database management
  • Subscription-based host list updates
  • Integration with the address book application
  • Efficient lookup operations

The BlockfileNamingService is the default NamingService, providing persistent storage and fast lookups for hostnames within the I2P network.
