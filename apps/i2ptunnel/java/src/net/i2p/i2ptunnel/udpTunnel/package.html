UDP tunnel implementation for I2PTunnel.

This package provides UDP tunneling functionality for I2PTunnel, allowing UDP-based applications to communicate over the I2P network. It handles the conversion between UDP packets and I2P messages, maintaining the semantics of UDP communication while providing I2P's anonymity and privacy benefits.

Features include:

  • UDP packet encapsulation in I2P messages
  • Connection tracking and state management
  • Port mapping and forwarding
  • Performance optimization for real-time traffic
  • Error handling and packet loss recovery
