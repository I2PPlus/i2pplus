SOCKS protocol tunnel implementation for I2P, supporting SOCKS 4, 4a, and 5. Allows applications using SOCKS proxies to communicate through the I2P network.

This package provides:

  • SOCKS 4 proxy server implementation
  • SOCKS 4a proxy with DNS resolution support
  • SOCKS 5 proxy with authentication support
  • Connection routing through I2P tunnels
  • Protocol-compliant SOCKS handling
