Definition-based incoming connection filter for I2PTunnel, allowing fine-grained access control for server tunnels. The filter definition format is documented in DefinitionParser.

This package provides:

  • Access rule parsing and validation
  • Connection filtering based on I2P destinations
  • Whitelist and blacklist support
  • Pattern-based destination matching
  • Flexible rule evaluation engine

Enables administrators to control which peers may connect to I2P server tunnels, enhancing security and control over published services.
