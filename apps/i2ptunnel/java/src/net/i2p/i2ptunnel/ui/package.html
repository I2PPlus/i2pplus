Public API classes for programmatic tunnel configuration and management. Separated from net.i2p.i2ptunnel.web in 0.9.19 to provide a stable API for external applications, including the Android client.

This package contains:

  • Tunnel configuration classes and builders
  • Tunnel type definitions and options
  • Configuration validation utilities
  • API for creating and managing tunnels programmatically

Maintained as a stable public API.
