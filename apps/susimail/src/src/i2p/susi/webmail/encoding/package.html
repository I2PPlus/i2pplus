Character encoding and MIME handling for SusiMail.

This package provides comprehensive support for handling various character encodings and MIME types in email messages. It ensures proper display and processing of international email content according to internet standards.

Features include:

  • Character set detection and conversion
  • MIME header parsing and encoding
  • Base64 and quoted-printable encoding/decoding
  • International character support (UTF-8, ISO-8859-*, etc.)
  • Content-Type and Content-Transfer-Encoding handling

This package ensures that SusiMail can properly handle email content in any language and encoding, providing a truly international email experience over I2P.
