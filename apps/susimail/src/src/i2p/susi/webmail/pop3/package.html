POP3 protocol implementation for SusiMail.

This package provides a complete implementation of the Post Office Protocol version 3 (POP3) as defined in RFC 1939. It allows SusiMail to retrieve email messages from POP3 mail servers over the I2P network.

Key features:

  • Full POP3 command implementation
  • Secure authentication mechanisms (APOP, USER/PASS)
  • Message listing and retrieval
  • Message deletion and status management
  • SSL/TLS support for secure connections
  • Error handling and protocol compliance

The implementation is designed to work seamlessly with I2P's SOCKS proxy infrastructure, providing secure and anonymous email retrieval.
