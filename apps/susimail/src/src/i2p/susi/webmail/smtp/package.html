SMTP protocol implementation for SusiMail.

This package provides a complete implementation of the Simple Mail Transfer Protocol (SMTP) as defined in RFC 5321 and related specifications. It enables SusiMail to send email messages through SMTP servers over the I2P network.

Key features:

  • Full SMTP command implementation (EHLO, MAIL, RCPT, DATA, etc.)
  • ESMTP extensions and capability negotiation
  • Authentication mechanisms (PLAIN, LOGIN, CRAM-MD5)
  • STARTTLS for secure connections
  • Message formatting and transmission
  • Pipelining and performance optimizations
