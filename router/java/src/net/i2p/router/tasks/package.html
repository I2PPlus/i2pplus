Periodic maintenance tasks and background jobs for I2P router operations.

This package contains various maintenance tasks, background jobs, and periodic operations that keep the I2P router running smoothly. These tasks handle everything from network exploration and peer management to statistics collection and system maintenance.

Key Components:

  • CoalesceStatsEvent - Aggregates and processes statistics
  • CryptoChecker - Checks cryptographic algorithm availability
  • GracefulShutdown - Coordinates clean router shutdown
  • InstallUpdate - Handles software update installation
  • MarkLiveliness - Periodic liveliness marker for multi-instance detection
  • OOMListener - Handles out-of-memory conditions
  • ReadConfigJob - Reloads router configuration
  • Republish - Periodically publishes router info
  • Restarter - Performs soft router restart
  • RouterWatchdog - Monitors router health and performance
  • ThreadDump - Generates thread dumps for diagnostics

Main Functionality:

  • Periodic statistics aggregation and coalescing
  • Router health monitoring and watchdog recovery
  • Cryptographic algorithm availability verification
  • Configuration reloading and updates
  • Graceful shutdown and restart coordination
  • Software update installation
  • Router info persistence and publication
  • Thread dump generation for diagnostics
  • Memory and out-of-memory condition handling
  • Permission fixer for Windows

Task Types:

  • Jobs - Scheduled tasks in the router job queue
  • Threads - Long-running background processes
  • TimedEvents - Periodic timer-based operations
  • One-time tasks - Single execution maintenance tasks

Usage Notes:

  • Internal router maintenance operations only
  • Not part of the public API
  • Essential for router stability and performance
  • Coordinated through Router.java task scheduler
