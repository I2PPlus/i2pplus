Post-quantum cryptographic algorithms for I2P.

This package provides implementations of post-quantum cryptographic algorithms designed to resist attacks from quantum computers. These algorithms are intended for future-proofing I2P's cryptographic infrastructure against advances in quantum computing.

Supported algorithms include:

  • Lattice-based cryptography (Kyber, Dilithium)
  • Code-based cryptography
  • Hash-based signatures
  • Multivariate cryptography
