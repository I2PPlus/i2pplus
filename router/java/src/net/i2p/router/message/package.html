Garlic message creation, parsing, and source routing for I2P communications.

This package provides the core functionality for creating, parsing, and managing garlic messages, which are the fundamental building blocks of I2P's anonymous communication system. Garlic messages allow for the encryption and routing of multiple messages together, providing both anonymity and message bundling capabilities.

Key Components:

  • GarlicMessageBuilder - Constructs garlic messages with multiple cloves
  • GarlicMessageParser - Parses and validates incoming garlic messages
  • GarlicMessageHandler - Processes and routes garlic messages
  • GarlicMessageReceiver - Receives and handles garlic message delivery
  • GarlicConfig - Configuration for garlic message creation
  • PayloadGarlicConfig - Configuration for payload-specific garlic messages
  • OutboundClientMessageJobHelper - Manages outbound client message jobs
  • OutboundClientMessageOneShotJob - Single-shot outbound message handling
  • SendMessageDirectJob - Direct message sending functionality
  • HandleGarlicMessageJob - Job for processing garlic messages
  • OutboundCache - Caching for outbound message optimization

Main Functionality:

  • Garlic message construction and encryption
  • Source routing for anonymous message delivery
  • Message parsing and validation
  • Outbound message job management
  • Direct message sending capabilities
  • Message caching and optimization
  • Clove management and processing

Usage Notes:

  • Essential for I2P's anonymous routing system
  • Provides the foundation for end-to-end encryption
  • Supports both client and router message types
  • Integrates with the router's job queue system
