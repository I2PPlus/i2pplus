Offline analysis tools for detecting Sybil attacks and malicious peer coordination in the I2P network.

This package provides:

  • Sybil attack detection algorithms
  • Peer relationship analysis
  • Statistical correlation tools
  • Result storage and loading mechanisms
  • Analysis report generation

Sybil attacks occur when a single adversary controls multiple nodes in the network to compromise security and anonymity. This package provides offline tools to analyze peer relationships and detect potential Sybil activity.

Since 0.9.38. Moved from console to router in 0.9.49.
