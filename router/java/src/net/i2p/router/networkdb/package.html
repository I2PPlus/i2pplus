Network database management and coordination for the I2P distributed hash table.

This package provides high-level coordination and management classes for the I2P network database, which is a distributed hash table (DHT) that stores router information and lease sets. While most of the detailed implementation is in the kademlia subpackage, this package contains the coordination logic and job management for network database operations.

Key Components:

  • HandleDatabaseLookupMessageJob - Processes database lookup requests
  • PublishLocalRouterInfoJob - Manages publication of local router information

Main Functionality:

  • Coordination of database lookup operations
  • Local router information publishing and updates
  • Integration with the Kademlia DHT implementation
  • Job scheduling and management for network database tasks

Usage Notes:

  • Most detailed implementation is in the kademlia subpackage
  • This package provides the high-level job coordination
  • Database operations are handled through scheduled jobs
  • Integrates with the router's job queue system
