Time synchronization and clock management for I2P router using NTP.

This package provides:

  • NTP client for time synchronization
  • Offset calculation and drift correction
  • Stratum level tracking
  • Skew detection and adjustment
  • Clock stability monitoring

Accurate time synchronization is essential for I2P network operations, including tunnel management, message validation, and coordination with other routers. The time package ensures router clocks stay synchronized with network time sources.
