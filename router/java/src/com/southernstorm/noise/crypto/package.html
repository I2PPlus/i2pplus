Noise Protocol Framework implementation for I2P.

This package provides an implementation of the Noise Protocol Framework, a modern cryptographic protocol for secure peer-to-peer communication. The Noise framework provides flexible, secure key exchange and encryption patterns suitable for various networking scenarios.

Key features:

  • Handshake pattern implementations
  • Cipher state management
  • DH state handling
  • Symmetric key operations
  • Protocol composition and customization

This implementation is used by I2P for establishing secure communication channels with strong forward secrecy and resistance to various attacks.
