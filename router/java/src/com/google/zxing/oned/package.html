One-dimensional (1D) barcode detection, decoding, and encoding for ZXing.

This package contains classes for detecting, decoding, and encoding various 1D barcode formats including UPC, EAN, Code 39, Code 128, and other linear barcode symbologies.

Key Components:

  • UPCEANReader - UPC-EAN barcode reader implementation
  • UPCAReader - UPC-A barcode reader implementation
  • UPCEReader - UPC-E barcode reader implementation
  • UPCAReader - UPC-A barcode reader implementation
  • UPCReader - UPC-C barcode reader implementation
  • Code39Reader - Code 39 barcode reader implementation
  • Code128Reader - Code 128 barcode reader implementation
  • EAN13Reader - EAN-13 barcode reader implementation
  • ITFReader - Interleaved 2 of 5 barcode reader implementation
  • RSS14Reader - RSS-14 barcode reader implementation
  • RSSExpandedReader - RSS Expanded barcode reader implementation
  • MultiFormatOneDReader - Multi-format 1D barcode reader
  • OneDWriter - Base 1D barcode writer implementation
  • UPCEANWriter - UPC-EAN barcode writer implementation
  • UPCAWriter - UPC-A barcode writer implementation
  • UPCEReader - UPC-E barcode writer implementation
  • UPCAWriter - UPC-A barcode writer implementation
  • UPCWriter - UPC-C barcode writer implementation
  • Code39Writer - Code 39 barcode writer implementation
  • Code128Writer - Code 128 barcode writer implementation
  • EAN13Writer - EAN-13 barcode writer implementation
  • ITFWriter - Interleaved 2 of 5 barcode writer
  • RSS14Writer - RSS-14 barcode writer implementation
  • RSSExpandedWriter - RSS Expanded barcode writer implementation

Main Functionality:

  • 1D barcode pattern detection and localization
  • Data decoding with error correction
  • 1D barcode generation and encoding
  • Support for multiple 1D barcode formats
  • Pattern matching and scoring
  • Checksum calculation and verification
  • Optimized for performance and memory usage

Usage Notes:

  • Supports all standard 1D barcode formats
  • Optimized for performance and memory usage
  • Handles various barcode quality levels
  • Used by MultiFormatReader for high-level operations
