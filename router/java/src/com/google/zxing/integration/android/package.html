Android integration utilities for ZXing barcode and QR code scanning.

This package provides Android-specific integration utilities and helper classes for integrating ZXing barcode and QR code scanning functionality into Android applications.

Key Components:

  • IntentIntegrator - Integration helper for Android Intent-based scanning
  • IntentResult - Result wrapper for Intent-based scanning
  • IntentResultHandler - Handler for processing scan results
  • BarcodeEncoder - Barcode generation utilities for Android
  • BarcodeDecoder - Barcode decoding utilities for Android
  • CaptureActivity - Main camera capture activity
  • DecodeHandler - Handler for barcode decoding operations
  • DecodeThread - Background thread for decoding
  • ViewfinderView - Custom view for camera preview
  • CameraManager - Camera management utilities

Main Functionality:

  • Android Intent integration for barcode scanning
  • Camera preview and capture management
  • Background barcode decoding
  • Result handling and processing
  • Barcode generation for Android

Usage Notes:

  • Designed for Android applications
  • Provides easy integration with Android Intents
  • Supports both scanning and generation
  • Handles camera permissions and lifecycle
