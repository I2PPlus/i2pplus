Aztec barcode detection, decoding, and encoding for ZXing.

This package contains classes for detecting, decoding, and encoding Aztec barcodes, a 2D matrix symbology that offers high data density and robust error correction.

Key Components:

  • AztecReader - Main Aztec barcode reader implementation
  • AztecWriter - Aztec barcode generation and encoding
  • Decoder - Aztec data decoding implementation
  • Detector - Aztec pattern detection
  • AztecDetectorResult - Detection result container
  • AztecCode - Aztec code data structure
  • Encoder - Aztec data encoding implementation
  • HighLevelEncoder - High-level data encoding
  • SimpleToken - Token representation for encoding
  • BinaryToken - Binary token for encoding

Main Functionality:

  • Aztec barcode pattern detection and localization
  • Data decoding with Reed-Solomon error correction
  • Aztec barcode generation and encoding
  • Support for compact and full-range Aztec codes
  • Optimized encoding for different data types
  • Mode selection and compression

Usage Notes:

  • Used in transportation, healthcare, and industrial applications
  • Supports both compact and standard Aztec formats
  • Excellent error correction capabilities
  • Handles various data types and encodings
