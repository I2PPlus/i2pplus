DataMatrix barcode detection, decoding, and encoding for ZXing.

This package contains classes for detecting, decoding, and encoding DataMatrix barcodes, a 2D matrix symbology known for high data density in a small space.

Key Components:

  • DataMatrixReader - Main DataMatrix barcode reader implementation
  • DataMatrixWriter - DataMatrix barcode generation and encoding
  • Detector - DataMatrix pattern detection
  • Decoder - DataMatrix data decoding implementation
  • DataMatrixDetectorResult - Detection result container
  • DataMatrixDecoder - DataMatrix decoding logic
  • DataMatrixBitParser - Bit matrix parsing utilities
  • DataMatrixVersion - Version information and capabilities
  • DataMatrixSymbolInfo - Symbol information container
  • DefaultPlacement - Default bit placement patterns

Main Functionality:

  • DataMatrix barcode pattern detection and localization
  • Data decoding with error correction
  • DataMatrix barcode generation and encoding
  • Support for various DataMatrix sizes and versions
  • Bit placement and pattern generation
  • Error correction using Reed-Solomon algorithms

Usage Notes:

  • Used in electronics, healthcare, and manufacturing
  • Supports both square and rectangular DataMatrix formats
  • Excellent for small item marking
  • Handles various data encodings and character sets
