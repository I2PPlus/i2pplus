Multiple barcode and QR code processing utilities for ZXing.

This package provides utilities and classes for detecting and decoding multiple barcodes and QR codes in a single image or scan operation.

Key Components:

  • MultipleBarcodeReader - Interface for reading multiple barcodes
  • GenericMultipleBarcodeReader - General-purpose multiple barcode reader
  • ByQuadrantReader - Reader that processes image by quadrants
  • QRCodeMultiReader - Multi-QR code specific reader
  • MultipleBarcodeDetector - Detector for multiple barcodes
  • MultiDetector - Generic multi-barcode detector
  • MultiFormatReader - Multi-format barcode reader
  • MultiFormatWriter - Multi-format barcode writer

Main Functionality:

  • Detection of multiple barcodes in single images
  • Batch processing of barcode results
  • Region-based barcode detection
  • Multi-format support for various barcode types
  • Optimized algorithms for multiple barcode scenarios

Usage Notes:

  • Useful for documents with multiple barcodes
  • Supports various barcode formats simultaneously
  • Optimized for performance with multiple detections
  • Provides flexible detection strategies
