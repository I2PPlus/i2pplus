MaxiCode detection implementation for ZXing.

This package contains the core detection logic for finding and locating MaxiCode barcodes in images, including finder pattern detection and alignment pattern processing.

Key Components:

  • Detector - Main MaxiCode detector implementation
  • FinderPatternFinder - Finder pattern detection logic
  • AlignmentPattern - Alignment pattern representation
  • AlignmentPatternFinder - Alignment pattern detection
  • FinderPattern - Finder pattern representation
  • FinderPatternInfo - Finder pattern information container
  • PatternResult - Pattern detection results
  • DetectorResult - Detection result container

Main Functionality:

  • MaxiCode finder pattern detection
  • Alignment pattern detection and verification
  • Pattern matching and scoring
  • Module size estimation
  • Perspective transformation calculation
  • MaxiCode localization and bounding

Usage Notes:

  • Core detection engine for MaxiCode barcodes
  • Handles various image conditions and orientations
  • Robust against noise and distortion
  • Used by MaxiCodeReader for high-level operations
