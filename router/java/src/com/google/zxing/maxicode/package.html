MaxiCode barcode detection, decoding, and encoding for ZXing.

This package contains the main classes for MaxiCode barcode processing, including reader, writer, detector, decoder, and encoder implementations.

Key Components:

  • MaxiCodeReader - Main MaxiCode barcode reader implementation
  • MaxiCodeWriter - MaxiCode barcode generation and encoding
  • Decoder - MaxiCode data decoding implementation
  • Detector - MaxiCode pattern detection
  • MaxiCodeDecoder - MaxiCode decoding logic
  • MaxiCodeBitParser - Bit matrix parsing utilities
  • MaxiCodeVersion - Version information and capabilities
  • MaxiCodeSymbolInfo - Symbol information container
  • DefaultPlacement - Default bit placement patterns

Main Functionality:

  • MaxiCode barcode pattern detection and localization
  • Data decoding with error correction
  • MaxiCode barcode generation and encoding
  • Support for various MaxiCode sizes and versions
  • Bit placement and pattern generation
  • Error correction using Reed-Solomon algorithms
  • Multiple data mode support
  • Structured append handling

Usage Notes:

  • Primarily used in healthcare and pharmacy applications
  • Supports both standard and structured MaxiCode formats
  • Excellent error correction capabilities
  • Handles various data encodings and character sets
