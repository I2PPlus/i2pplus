MaxiCode decoding implementation for ZXing.

This package contains the core decoding logic for MaxiCode barcodes, including data extraction, error correction, and format information processing.

Key Components:

  • Decoder - Main MaxiCode decoder implementation
  • BitMatrixParser - Bit matrix parsing and data extraction
  • DataBlock - Data block management for error correction
  • DecodedBitStreamParser - Bit stream parsing and data decoding
  • MaxiCodeDecoderMeta - Metadata for MaxiCode decoding
  • ErrorCorrection - Error correction implementation
  • MaxiCodeBitParser - MaxiCode-specific bit parsing

Main Functionality:

  • MaxiCode data extraction and parsing
  • Error correction using Reed-Solomon algorithms
  • Bit stream parsing and data decoding
  • Format and version information decoding
  • Multiple data mode support
  • Structured append handling

Usage Notes:

  • Core decoding engine for MaxiCode barcodes
  • Handles all MaxiCode versions and error correction levels
  • Supports damaged MaxiCode recovery
  • Used by MaxiCodeReader for high-level operations
