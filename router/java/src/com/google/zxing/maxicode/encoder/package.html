MaxiCode encoding implementation for ZXing.

This package contains the core encoding logic for generating MaxiCode barcodes, including data encoding, error correction generation, and matrix construction.

Key Components:

  • Encoder - Main MaxiCode encoder implementation
  • HighLevelEncoder - High-level data encoding
  • DefaultPlacement - Default bit placement patterns
  • MaxiCodeEncoderContext - Context for encoding operations
  • MaxiCodeSymbolInfo - Symbol information container
  • MaxiCodeVersion - Version information and capabilities
  • MaxiCodeEncoder - MaxiCode encoding logic
  • MaxiCodePlacement - Bit placement implementation

Main Functionality:

  • MaxiCode data encoding and compression
  • Error correction code generation
  • Bit placement and pattern generation
  • Version and format information encoding
  • Optimized encoding for different data types
  • Mode selection and compression

Usage Notes:

  • Core encoding engine for MaxiCode generation
  • Supports all MaxiCode versions and error correction levels
  • Optimized for performance and memory usage
  • Used by MaxiCodeWriter for high-level operations
