PDF417 barcode detection, decoding, and encoding for ZXing.

This package contains classes for detecting, decoding, and encoding PDF417 barcodes, a 2D stacked linear symbology used in applications like ID cards and boarding passes.

Key Components:

  • PDF417Reader - Main PDF417 barcode reader implementation
  • PDF417Writer - PDF417 barcode generation and encoding
  • Detector - PDF417 pattern detection
  • Decoder - PDF417 data decoding
  • PDF417ResultMetadata - Metadata for PDF417 results
  • BarcodeValue - Barcode value container
  • ModulusGF - Galois Field arithmetic for error correction
  • PDF417CodewordDecoder - Codeword decoding logic
  • PDF417ScanningDecoder - Scanning-based decoding
  • PDF417DetectorResult - Detection result container

Main Functionality:

  • PDF417 barcode pattern detection and localization
  • Data decoding with error correction
  • PDF417 barcode generation and encoding
  • Support for various PDF417 variants
  • Error correction using Reed-Solomon algorithms
  • Metadata extraction (optional fields, file segments)

Usage Notes:

  • Used in ID documents, boarding passes, and shipping labels
  • Supports both standard and compact PDF417 formats
  • Handles damaged barcode recovery
  • Supports binary and text data encoding
