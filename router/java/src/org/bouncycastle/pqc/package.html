Post-Quantum Cryptography (PQC) package providing implementations of quantum-resistant cryptographic algorithms.

This package contains the main interfaces and classes for post-quantum cryptography, including key encapsulation mechanisms (KEM) and other quantum-resistant algorithms designed to be secure against attacks by quantum computers.

Key Components:

  • KEM (Key Encapsulation Mechanism) interfaces and implementations
  • Post-quantum algorithm parameters and specifications
  • Integration with BouncyCastle's cryptographic provider framework

Main Functionality:

  • Quantum-resistant key generation and management
  • Secure encapsulation and decapsulation operations
  • Algorithm parameter management for PQC algorithms
