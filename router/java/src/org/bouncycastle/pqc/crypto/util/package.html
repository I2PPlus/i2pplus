Utility classes for Post-Quantum Cryptography operations.

This package provides helper classes and utility methods for working with post-quantum cryptographic algorithms, particularly for key encapsulation mechanisms and related operations.

Key Components:

  • SecretWithEncapsulation implementation for KEM operations
  • Utility methods for key management and encapsulation
  • Helper classes for post-quantum cryptographic operations

Main Functionality:

  • Encapsulation of secret keys with ciphertext
  • Utility methods for KEM (Key Encapsulation Mechanism) operations
  • Support for post-quantum cryptographic key handling

Usage Notes:

  • These utilities are designed to work with BouncyCastle's post-quantum implementations
  • Provides standardized interfaces for key encapsulation operations
