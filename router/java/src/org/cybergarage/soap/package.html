SOAP (Simple Object Access Protocol) implementation for UPnP.

This package provides classes for handling SOAP messages and protocol operations used by the UPnP framework for service invocation and control operations.

Key Components:

  • SOAP - Main SOAP protocol constants and utilities
  • SOAPRequest - Represents SOAP request messages
  • SOAPResponse - Represents SOAP response messages

Main Functionality:

  • SOAP message parsing and generation
  • XML serialization for SOAP
  • HTTP transport for SOAP
  • SOAP action invocation support
  • Protocol compliance and utilities

Usage Notes:

  • Used by UPnP framework for SOAP-based control operations
  • Supports SOAP 1.1 and 1.2 specifications
  • Provides both client and server SOAP functionality
