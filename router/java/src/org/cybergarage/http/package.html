HTTP protocol implementation and utilities.

This package provides classes for handling HTTP requests, responses, and protocol operations used by the UPnP framework for HTTP-based communication.

Key Components:

  • HTTP - Main HTTP protocol constants and utilities
  • HTTPRequest - Represents HTTP request messages
  • HTTPResponse - Represents HTTP response messages
  • HTTPPacket - Base class for HTTP messages
  • HTTPHeader - Manages HTTP headers
  • HTTPServer - HTTP server implementation
  • Parameter - HTTP parameter handling

Main Functionality:

  • HTTP request and response parsing
  • HTTP header management
  • Parameter handling and parsing
  • HTTP server functionality
  • Protocol compliance and utilities

Usage Notes:

  • Used by UPnP framework for HTTP-based communication
  • Supports HTTP/1.1 protocol specification
  • Provides both client and server HTTP functionality
