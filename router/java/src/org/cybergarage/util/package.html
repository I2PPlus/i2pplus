General utility classes and helper functions.

This package provides utility classes for common operations used throughout the UPnP framework, including threading, synchronization, debugging, and file operations.

Key Components:

  • Debug - Debug logging and utilities
  • StringUtil - String manipulation and validation utilities
  • FileUtil - File operations and utilities
  • ListenerList - Event listener management
  • Mutex - Thread synchronization primitives
  • TimerUtil - Timer and scheduling utilities
  • ThreadCore - Thread management base class

Main Functionality:

  • String manipulation and validation
  • File I/O operations
  • Thread synchronization and management
  • Event listener patterns
  • Debug logging and tracing
  • Timer and scheduling utilities

Usage Notes:

  • Used throughout UPnP framework for common operations
  • Provides thread-safe utilities where needed
  • Supports both development and runtime debugging
