XML parser implementations and utilities.

This package provides concrete implementations of XML parsers, including a JAXP-based parser for processing XML documents used by the UPnP framework.

Key Components:

  • JaxpParser - JAXP-based XML parser implementation
  • Parser - XML parser interface
  • ParserException - XML parsing error handling

Main Functionality:

  • XML document parsing using JAXP
  • DOM-style XML tree construction
  • XML validation and error reporting
  • Parser factory and abstraction

Usage Notes:

  • Used by UPnP framework for parsing device descriptions
  • Provides JAXP-compliant XML parsing
  • Supports both validation and non-validating parsing modes
