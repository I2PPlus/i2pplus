XML parsing and manipulation utilities.

This package provides classes for XML document parsing, node manipulation, and XML processing utilities used by the UPnP framework for handling XML-based descriptions and messages.

Key Components:

  • XML - Main XML utility class and constants
  • Node - Represents XML document nodes
  • NodeList - Manages collections of XML nodes
  • Attribute - Represents XML attributes
  • AttributeList - Manages collections of XML attributes
  • Parser - XML parser interface and factory
  • ParserException - XML parsing error handling

Main Functionality:

  • XML document parsing and creation
  • Node traversal and manipulation
  • Attribute management and access
  • XML validation and error handling
  • Parser abstraction and factory pattern

Usage Notes:

  • Used by UPnP framework for XML description processing
  • Provides DOM-like XML manipulation interface
  • Supports both parsing and generation of XML documents
