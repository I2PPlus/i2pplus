Simple Service Discovery Protocol (SSDP) implementation for UPnP.

This package provides classes for handling SSDP operations, including device discovery, search responses, and notification management for UPnP networks.

Key Components:

  • SSDPNotifySocket - Handles SSDP NOTIFY messages for device advertisements
  • SSDPSearchResponseSocket - Manages SSDP search response operations
  • SSDPResponse - Represents SSDP response messages
  • HTTPUSocket - UDP socket implementation for SSDP multicast
  • SSDPSearchResponseSocketList - Manages multiple search response sockets
  • SSDPNotifySocketList - Manages multiple notification sockets

Main Functionality:

  • SSDP multicast message handling
  • Device discovery and advertisement
  • Search request and response management
  • UDP socket management for SSDP operations
  • Multicast group communication

Usage Notes:

  • Implements the discovery layer of UPnP protocol
  • Uses UDP multicast on 239.255.255.250:1900
  • Handles both M-SEARCH and NOTIFY SSDP messages
