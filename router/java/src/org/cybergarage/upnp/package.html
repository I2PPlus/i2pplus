Universal Plug and Play (UPnP) framework core components.

This package provides the main classes and interfaces for implementing UPnP devices and control points. It includes the core functionality for device discovery, service description, and control operations.

Key Components:

  • Device - Represents UPnP devices with services and embedded devices
  • Service - Provides UPnP service functionality with actions and state variables
  • Action - Represents invocable operations on UPnP services
  • ControlPoint - Manages UPnP device discovery and control
  • Argument - Defines parameters for UPnP actions

Main Functionality:

  • UPnP device discovery and advertisement
  • Service description and invocation
  • Event notification and subscription management
  • Device and service state management

Usage Notes:

  • This is the core UPnP framework used by I2P for NAT traversal
  • Provides both device and control point capabilities
  • Supports UPnP 1.0 specification
