XML parsing and data handling for UPnP device descriptions.

This package provides classes for parsing and managing XML data structures used in UPnP device and service descriptions. It handles the conversion between XML and Java objects.

Key Components:

  • DeviceData - XML data container for UPnP device information
  • ServiceData - XML data container for UPnP service information
  • ActionData - XML data container for UPnP action definitions
  • StateVariableData - XML data container for state variable definitions
  • ArgumentData - XML data container for action argument definitions
  • NodeData - Generic XML node data wrapper

Main Functionality:

  • XML parsing and serialization for UPnP descriptions
  • Data structure conversion between XML and Java objects
  • Device and service metadata management
  • UPnP description document processing

Usage Notes:

  • Used internally by the UPnP framework for description processing
  • Provides abstraction layer over XML parsing operations
