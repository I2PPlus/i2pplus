UPnP control and action invocation framework.

This package provides classes for handling UPnP control requests, action invocation, and response management. It implements the control side of the UPnP architecture.

Key Components:

  • Control - Main control point for UPnP operations
  • ActionRequest - Represents UPnP action invocation requests
  • ActionResponse - Handles UPnP action invocation responses
  • QueryRequest - Represents state variable query requests
  • QueryResponse - Handles state variable query responses
  • RenewSubscriber - Manages subscription renewal

Main Functionality:

  • UPnP action invocation and execution
  • State variable querying and monitoring
  • Control request and response handling
  • Subscription management and renewal

Usage Notes:

  • Used by UPnP control points to interact with devices
  • Handles both SOAP-based control requests and responses
  • Provides asynchronous operation support
