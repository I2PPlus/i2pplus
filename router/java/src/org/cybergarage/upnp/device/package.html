UPnP device implementation and management.

This package provides classes for implementing UPnP devices, including device advertisement, discovery response handling, and device lifecycle management.

Key Components:

  • Advertiser - Handles UPnP device advertisement and announcements
  • Description - Manages UPnP device description documents
  • Disposer - Handles device cleanup and resource disposal
  • SearchListener - Listens for UPnP search requests
  • PresentationListener - Handles device presentation URL requests
  • DeviceChangeListener - Notifies of device state changes
  • MAN, NTS, NT, ST, USN - UPnP protocol constants and enums

Main Functionality:

  • UPnP device advertisement and discovery
  • Device description management
  • Search request handling
  • Device lifecycle and state management
  • Presentation URL handling

Usage Notes:

  • Used by UPnP devices to advertise their presence
  • Handles SSDP (Simple Service Discovery Protocol) operations
  • Manages device registration and deregistration
