UPnP event notification and subscription management.

This package provides classes for handling UPnP event notifications, including subscription management, event delivery, and property change notifications.

Key Components:

  • Subscription - Manages UPnP event subscriptions
  • Subscriber - Represents a UPnP event subscriber
  • Property - Represents UPnP state variable properties
  • NotifyRequest - Handles event notification requests
  • SubscriptionRequest - Manages subscription requests
  • SubscriptionResponse - Handles subscription responses
  • EventListener - Interface for event notification handling

Main Functionality:

  • UPnP event subscription management
  • Property change notifications
  • Event message delivery
  • Subscription lifecycle management
  • GENA (Generic Event Notification Architecture) support

Usage Notes:

  • Implements the eventing side of UPnP architecture
  • Uses HTTP callbacks for event delivery
  • Supports both subscription and unsubscription operations
