Noise Protocol Framework protocol patterns and state machines.

This package contains the high-level protocol patterns and state machines for the Noise Protocol Framework. It defines the various handshake patterns and manages the protocol state throughout the secure communication process.

Components include:

  • Handshake pattern definitions
  • Protocol state machines
  • Message processing and generation
  • Transport layer integration
  • Error handling and validation

These patterns provide the foundation for secure key exchange and encrypted communication in I2P, offering various security properties and performance characteristics suitable for different use cases.
