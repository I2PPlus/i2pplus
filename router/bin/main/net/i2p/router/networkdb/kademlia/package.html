Kademlia DHT implementation and floodfill router functionality for I2P.

This package contains the complete implementation of the I2P network database based on the Kademlia distributed hash table algorithm. It provides the core functionality for storing and retrieving router information and lease sets, with special support for floodfill routers that act as super-nodes in the network.

Key Components:

  • FloodfillNetworkDatabaseFacade - Floodfill router implementation
  • KademliaNetworkDatabaseFacade - Standard Kademlia DHT implementation
  • DataStore - Persistent storage for network database entries
  • SearchJob - Database lookup and search operations
  • StoreJob - Database storage operations
  • ExploreJob - Network exploration and discovery
  • LookupThrottler - Rate limiting for database operations
  • PersistentDataStore - Long-term storage management
  • PeerSelector - Router selection algorithms

Main Functionality:

  • Kademlia DHT routing and maintenance
  • Floodfill router operations and responsibilities
  • Database lookup, search, and storage operations
  • Network exploration and peer discovery
  • Rate limiting and throttling of database operations
  • Persistent storage and caching of network data
  • Router selection and peer management

Usage Notes:

  • Floodfill routers provide enhanced database services
  • Standard routers use the basic Kademlia implementation
  • All database operations are performed through jobs
  • Implements the I2P network database specification
