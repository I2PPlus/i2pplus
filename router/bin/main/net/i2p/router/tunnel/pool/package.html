Tunnel pool management, creation, and peer selection for I2P's anonymous routing system.

This package provides:

  • Tunnel creation and build message processing
  • Peer selection algorithms for tunnel participants
  • Inbound and outbound tunnel pool management
  • Adaptive tunnel build strategies
  • Build request throttling and acceptance policies
  • Tunnel pool configuration and optimization

Manages the lifecycle of tunnel pools, handling build requests based on local statistics and network conditions. Implements intelligent peer selection to optimize tunnel performance and anonymity.
