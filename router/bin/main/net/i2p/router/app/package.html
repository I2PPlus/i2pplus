Interface and framework for applications managed by the I2P router via clients.config. Classes implementing the RouterApp interface are controlled through this interface rather than using main().

This package provides:

  • RouterApp interface for managed applications
  • Application lifecycle management (startup, shutdown)
  • Integration with router context and services
  • Inter-application communication capabilities

Benefits for clients using this interface:

  • Access to current context via constructor
  • Complete lifecycle management by router
  • No need for static references
  • Ability to discover and communicate with other clients
