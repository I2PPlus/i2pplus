Transport layer implementations and management for I2P network communication.

This package provides the transport system that enables I2P to communicate over various underlying network protocols. The transport layer sits below the I2P routing layer and above the network stack, providing abstraction and management of different communication methods.

Key Components:

  • TransportManager - Coordinates all transport protocols
  • Transport - Base interface for transport implementations
  • CommSystemFacade - High-level communication system interface
  • TransportImpl - Base implementation for transports
  • TransportEventListener - Handles transport events
  • TransportBidirectional - Bidirectional transport support
  • TransportEstablishment - Connection establishment management

Supported Transport Types:

  • NTCP2 - New TCP transport with improved security and performance
  • SSU - Secure UDP transport for firewall traversal
  • HTTP - Optional HTTP transport for restrictive networks
  • Custom transports - Extensible framework for new protocols

Main Functionality:

  • Transport protocol management and coordination
  • Connection establishment and maintenance
  • Message delivery over various protocols
  • Transport selection and failover
  • Bandwidth management and throttling
  • Connection monitoring and statistics
  • Transport-specific encryption and authentication
  • NAT traversal and firewall compatibility

Usage Notes:

  • Provides abstraction over underlying network protocols
  • Supports multiple simultaneous transports
  • Essential for network connectivity and peer communication
  • Integrates with tunnel and routing systems
