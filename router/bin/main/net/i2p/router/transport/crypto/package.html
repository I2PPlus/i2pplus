Cryptographic operations for I2P transport protocols, including key exchange algorithms, session key management, and transport-layer encryption.

This package provides:

  • Diffie-Hellman key exchange for transport establishment
  • Session key generation and management
  • Transport-specific encryption implementations
  • Cryptographic material for NTCP and SSU transports
  • Key agreement protocols for peer authentication

Essential for establishing secure, authenticated communication channels between I2P routers over various transport protocols.
